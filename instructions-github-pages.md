# Publier la démo avec GitHub Pages

GitHub Pages est une bonne alternative gratuite à Netlify pour cette démo.

## Étapes

1. Va sur https://github.com
2. Crée un compte gratuit ou connecte-toi.
3. Clique sur le bouton + en haut à droite, puis New repository.
4. Nom du dépôt : agriculture-fidelite
5. Mets le dépôt en Public.
6. Clique sur Create repository.
7. Clique sur uploading an existing file.
8. Dépose les fichiers du dossier agriculture-fidelite :
   - index.html
   - logo-agriculture.jpg
   - manifest.json
   - service-worker.js
   - .nojekyll
9. Clique sur Commit changes.
10. Va dans Settings.
11. Dans le menu de gauche, clique Pages.
12. Dans Build and deployment :
    - Source : Deploy from a branch
    - Branch : main
    - Folder : / root
13. Clique Save.
14. Attends 1 à 3 minutes.
15. GitHub donnera un lien du type :
    https://TON-NOM.github.io/agriculture-fidelite/

## Sur téléphone

Ouvre le lien sur ton téléphone.

Sur iPhone : Safari > Partager > Sur l'écran d'accueil.
Sur Android : Chrome > menu > Ajouter à l'écran d'accueil / Installer.

## Si tu ne vois pas Scanner ou Autres administrateurs

C'est le cache de l'ancienne version.

- Supprime l'ancienne icône de l'écran d'accueil.
- Ouvre le lien dans le navigateur.
- Recharge la page.
- Ajoute de nouveau à l'écran d'accueil.

Code admin par défaut : 1900
